# Collaborative Docs & Notes Platform

A real-time collaborative document editor built using the MERN stack. This application allows multiple users to create, edit, and collaborate on documents simultaneously with live synchronization.

---

## 🚀 Features

- Real-time multi-user document editing
- Live synchronization using Socket.IO
- Rich text editor with formatting support using Quill
- Auto-save functionality
- Shareable document links
- Responsive and user-friendly interface
- Persistent document storage with MongoDB

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Quill Editor
- Tailwind CSS
- Socket.io-client

### Backend
- Node.js
- Express.js
- Socket.IO
- REST API

### Database
- MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Saqib-Bahadur/collaborative-docs-platform.git
cd collaborative-docs-platform
```

---

### 2️⃣ Install Dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Run the Application

#### Start Backend

```bash
cd backend
npm start
```

#### Start Frontend

```bash
cd frontend
npm run dev
```
